# Einrichten eines Jupyterhubservers auf der Basis von Debian Trixie und Conda

Dieses Github-Projekt beschreibt, wie man einen Jupyterhubserver auf der Basis von Debian Trixie und Conda für ein schulisches Intranet einrichten kann. Der Server soll eine Benutzerauthentifizierung besitzen, so dass jeder Schüler seine eigene Jupyterlabumgebung vorfindet. Der Adminuser kann Nutzer im Linuxsystem automatisch mit Hilfe von Shellscripten anlegen. Es ist Shellscripten auch möglich, Dateien in die Nutzerverzeichnisse zu verteilen bzw. sie wieder einzusammeln. Weiterhin soll es möglich sein, für alle Nutzer zusätzliche Pytthonpakete (z.B. ipycanvas, matplotlib ...) installieren.
